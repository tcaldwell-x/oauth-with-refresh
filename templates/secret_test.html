Test Workflow
  1. Save tokens: Log in with each test account, then click "Save for Rotation Test" on the token page. Log out between accounts.
  2. Test API calls (before rotation): Click Test API Call for each user below to confirm they work.
  3. Rotate the secret: Go to the Developer Console and regenerate your client secret.
  4. Update the app: Set the new X_CLIENT_SECRET in your .env and restart the server.
  5. Test again: Come back here and click Test API Call & Test Refresh for each user. The results will tell you whether rotation broke anything.
